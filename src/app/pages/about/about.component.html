Seeds of Code

These three key experiences led me to my coding journey:

  1. 1. Working as an SEO content manager in e-commerce, I saw firsthand the results of coders' work and spent time alongside them.
  2. 2. Managing content led me to WordPress, and to satisfy my curiosity, I began learning website creation through boluda.com.
  3. 3. At an Excel training, I was amazed by a demonstration of how to automate tasks with macros and VBA. Seeing how a simple macro could transform a sad emoji into a happy one and how little code was needed to achieve results by cleaning up the VBA script ignited my interest in programming logic.

Hitting the Code

This sparked my exploration of VBA and Apps Scripts. Simultaneously, I wanted to break free from WordPress limitations and build custom websites, so I began learning web development fundamentals.

I explored Responsive Web Design at freeCodeCamp, HTML and CSS at boluda.com, Full Stack Web Development at Mimo.org, w3school courses and various other online resources.

I enrolled in a Full Stack Web Developer Bootcamp at CodeFactory in Vienna, significantly expanding my front-end knowledge and gaining back-end skills.

I've been coding consistently since then. While working on various projects, I've explored CSS animations and Tailwind CSS, but my primary focus has been Angular. Next, I plan to delve deeper into Symfony.
